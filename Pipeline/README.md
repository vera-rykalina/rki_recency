# Recency Pipeline (phyloTSI)
This is a Nextflow pipeline based on phyloTSI method (by T. Golubchik). The pipeline includes the following tools:
- kraken2
- shiver
- kallisto
- iva
- phyloscanner
- iqtree
- phyloTSI
